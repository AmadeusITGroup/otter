## Description

The module provides in-house eslint plugins to use in your own eslint configuration.

## How to install

```shell
ng add @o3r/eslint-plugin
```

> [!WARNING]
> This module requires [@o3r/core](https://www.npmjs.com/package/@o3r/core) to be installed.
