# Otter AI Primitives Hub bundle

## How to install

1. Install the VSCode extension [AI Primitives Hub](https://marketplace.visualstudio.com/items?itemName=AmadeusITGroup.ai-primitives-hub).
2. Open VS Code Command Palette (`Ctrl+Shift+P`)
3. Run: "AI Primitives Hub: Add Source"
4. Select "Collection from GitHub repository"
5. Enter this repo URL: `https://github.com/AmadeusITGroup/otter`
6. You can install the `Otter` bundle from the AI Primitives Hub marketplace
