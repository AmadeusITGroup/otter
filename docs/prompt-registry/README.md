# Otter prompt registry bundle

## How to install

1. Install the VSCode extension [Prompt Registry](https://marketplace.visualstudio.com/items?itemName=AmadeusITGroup.prompt-registry).
2. Open VS Code Command Palette (`Ctrl+Shift+P`)
3. Run: "Prompt Registry: Add Source"
4. Select "Collection from GitHub repository"
5. Enter this repo URL: `https://github.com/AmadeusITGroup/otter`
6. You can install the `Otter` bundle from the Prompt Registry marketplace
