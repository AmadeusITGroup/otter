Generate your first SDK

Specifications

The first step in the generation of an SDK is the defintition of the specifications. These can be written in a file of type JSON or YAML.

The Otter Showcase SDK has been created with the specifications made available by the Swagger Petstore repository. You can find these specifications in the openapi.yml of the repository, or on the right on this page.

You can get to know these specifications and eventually copy them in order to generate your own SDK locally, which will be explained in the following steps.
