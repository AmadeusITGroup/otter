Let's manage a set of apis

Introduction:

When dealing with an Angular project, you will need to ensure that your ApiClient will be shared accross your application. Otter framework provides the ApiManager service to manage your api collections.

Prerequisite:

- Install the @o3r/apis-manager in the project with npm install @o3r/apis-manager.

Objectives:

Leverage the ApiManager service to access two different clients to retrieve the list of available pets and submit an order for the first pet returned.

Exercise

Integrate the ApiManager in your application module. You can inspire yourself with the following lines: 

    
      const apiConfig: ApiClient = new ApiFetchClient(
        {
          basePath: 'https://petstore3.swagger.io/api/v3',
          requestPlugins: [],
          fetchPlugins: [],
          logger
        }
      );
      return new ApiManager(apiConfig, {
        
      });

      export const appConfig: ApplicationConfig = {
        providers: [],
        imports: [ApiManagerModule.forRoot()]
      };
