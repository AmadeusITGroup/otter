Generate your first SDK

Command lines

There are several command lines available to generate an SDK depending on the use cases. These cases are covered in the documentation of the package @ama-sdk/schematics.

In this training, you will be creating a new SDK repository. To do so, you can run the command line displayed in the example on the right. When running this command, you should expect all of the generated files to be similar to those displayed on the right.
